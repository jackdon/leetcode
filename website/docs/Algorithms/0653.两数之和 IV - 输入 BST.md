给定一个二叉搜索树和一个目标结果，如果 BST 中存在两个元素且它们的和等于给定的目标结果，则返回 true。

**案例 1:**


<pre><br/><strong>输入:</strong> <br/>    5<br/>   / \<br/>  3   6<br/> / \   \<br/>2   4   7<br/><br/>Target = 9<br/><br/><strong>输出:</strong> True<br/></pre>
