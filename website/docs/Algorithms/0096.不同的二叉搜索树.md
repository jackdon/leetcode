title: 0096.不同的二叉搜索树
---
给定一个整数 _n_，求以 1 ... _n_ 为节点组成的二叉搜索树有多少种？

**示例:**


<pre><strong>输入:</strong> 3<br/><strong>输出:</strong> 5<br/><strong>解释:<br/></strong>给定 <em>n</em> = 3, 一共有 5 种不同结构的二叉搜索树:<br/><br/>   1         3     3      2      1<br/>    \       /     /      / \      \<br/>     3     2     1      1   3      2<br/>    /     /       \                 \<br/>   2     1         2                 3</pre>
