title: 递归乘法
---
递归乘法。 写一个递归函数，不使用 * 运算符， 实现两个正整数的相乘。可以使用加号、减号、位移，但要吝啬一些。

 **示例1:**


<pre><br/><strong> 输入</strong>：A = 1, B = 10<br/><strong> 输出</strong>：10<br/></pre>

 **示例2:**


<pre><br/><strong> 输入</strong>：A = 3, B = 4<br/><strong> 输出</strong>：12<br/></pre>

 **提示:**

- 保证乘法范围不会溢出
